1. Introduction

NTUplace is a VLSI placement tool for large-scale mixed-size designs.
It was developed at the Graduate Institute of Electronics Engineering, National Taiwan University, Taipei, Taiwan.

3. Program Usage

-noglobal Turn off the global placement.
-nolegal Turn off the look-ahead legalization, legalization, detailed placement.
-nodetail Turn off the detailed placement.
-loadpl filename.pl Load the initial placement from the file.
-util utilization Set placement utilization.
-out prefix Output result in prefix.ntup.pl
-plt Plot the placement only.
-MRT Turn on macro rotation [6]

Examples:

./NTUplace -aux ibm01.aux

Run NTUplace for the circuit ibm01. Placement result is saved in ibm01.ntup.pl. GNUplot figure is saved in ibm01.ntup.plt.

./NTUplace -aux ibm01.aux -loadpl gp.pl -noglobal Legalize and detailed place the circuit ibm01. Global placement is read from the file "gp.pl".
./NTUplace -aux ibm01.aux -loadpl result.pl -plt Load ibm01 placement from "result.pl" and plot the figure.
./NTUplace -aux adaptec1.aux -util 0.8 Place the circuit adaptec1 using target utilization 0.8.
